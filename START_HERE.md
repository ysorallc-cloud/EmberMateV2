## 💡 Quick Tips


### Tip 2: Customize Colors
Edit CSS custom properties in `landing-styles.css`:
```css
:root {
    --primary-teal: #2D9B9B;  /* Your color */
}
```

### Tip 3: Deploy to Vercel

```bash
vercel --prod
```
