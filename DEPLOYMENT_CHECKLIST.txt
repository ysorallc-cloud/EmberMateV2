STEP 1: Download All Files
---------------------------
Click each link below to download:

1. index.html (landing page)
2. landing-styles.css
3. landing.js
4. auth.js
5. auth.css
6. encryption.js
7. README.md

Save all files to your EmberMateV2 folder.


STEP 2: Rename Your App File
-----------------------------
If you have an existing index.html with your app:
• Rename it to: app.html
• The new index.html will be your landing page


STEP 3: Add Integration Code to app.html
-----------------------------------------
Open app.html and add to <head> section:

<script src="https://cdnjs.cloudflare.com/ajax/libs/crypto-js/4.1.1/crypto-js.min.js"></script>
<link rel="stylesheet" href="auth.css">
<script src="auth.js" defer></script>
<script src="encryption.js" defer></script>

See README.md for complete integration HTML.


STEP 4: Push to GitHub
-----------------------
cd EmberMateV2
git add .
git commit -m "Add landing page, auth, and encryption"
git push


STEP 5: Enable GitHub Pages
----------------------------
1. Go to: https://github.com/ysorallc-cloud/EmberMateV2/settings/pages
2. Source: main branch
3. Click Save
4. Wait 2-3 minutes
