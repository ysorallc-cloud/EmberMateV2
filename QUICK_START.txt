## Get Started in 3 Steps

### Step 1: Choose Your Starting Point

**Option A: View the Landing Page** (Recommended for first-time users)
```bash
# Open landing.html in your browser
open landing.html

# Or on Windows
start landing.html

# Or on Linux
xdg-open landing.html
```

**Option B: Jump Straight to the App**
```bash
# Open index.html in your browser
open index.html
```

### Step 2: Use a Local Server (Optional but Recommended)

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js
npx http-server -p 8000

# Using npm scripts (if you have package.json)
npm start          # Opens landing page
npm run start:app  # Opens main app
```

Then visit: `http://localhost:8000/landing.html`

### Step 3: Deploy (When Ready)

```bash
# Deploy to Vercel
npm run deploy

# Or manually
vercel --prod
```

### Change Colors (5 minutes)


## ğŸ› Troubleshooting

**Problem:** Landing page looks broken
**Solution:** Make sure `landing-styles.css` is in the same directory

**Problem:** Main app doesn't work
**Solution:** Check browser console for errors, ensure `app.js` is loaded

**Problem:** Data doesn't persist
**Solution:** Check if cookies/LocalStorage are enabled in browser

**Problem:** Deployment fails
**Solution:** Check `vercel.json` is present and valid
