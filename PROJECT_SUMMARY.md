## 🚀 Three Ways to Deploy

### Method 1: Automatic (Easiest) ⭐

```bash
cd embermate
bash deploy.sh
# Follow the prompts!
```

### Method 2: Manual (Most Control)

```bash
cd embermate
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/embermate.git
git push -u origin main
```

Then deploy on [vercel.com](https://vercel.com)

### Method 3: Drag & Drop (Simplest)

1. Go to your GitHub repository
2. Upload all files
3. Go to Vercel
4. Import repository
5. Deploy!
