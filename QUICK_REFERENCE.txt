┌──────────────────────────────────────────────────────────────────┐
│ 🔧 TROUBLESHOOTING                                               │
└──────────────────────────────────────────────────────────────────┘

Deployment Failed?
→ Check: vercel logs [url]
→ Validate: jq empty vercel.json
→ Retry: vercel --prod --force

Site Not Loading?
→ Wait 2 minutes (DNS)
→ Clear browser cache
→ Check Vercel dashboard
→ Test with different browser

Changes Not Showing?
→ Verify: git log --oneline -n 1
→ Force deploy: vercel --prod --force
→ Clear CDN cache: Wait 1-2 minutes
